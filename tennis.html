Tennis league tournament

#1. What is this project about?

The aim for this project is to create tennis tournaments. The idea came to me when I identified the need to speed up tennis tournaments in tennis clubs. Usually, these tournaments lack of a system which brings that the draw of the tournament have to be written on paper and then being published on the board. Nothing is going to the record and tennis players have no statistics of their performance.Finally, there is a big problem every time for the registration, this is mainly because there is no a centralized place where players can enroll the tournament. This web project was in collaboration with 1 IT developer.

The goal for the project was to create a web application where tennis players can see the statistics for their performance, see their next matches and find a way to feel motivated.

#2. Features on this project

The system provides all the functionalities to create the tournament in a very smooth way. The tournament manager will see the people who are registered. When it is time to begin the tournament, he will put a name for the tournament and choose the type of tournament. The algorithm provides 2 playing modalities; Round Robin and Playoff mode.

#3. Proof of concept

This project was running in January 2018 in Mexico. There were 4 tournaments in total. The first tournamente had 13 participants. This number increased during the time and the last tournament had a total of 20 participants.
